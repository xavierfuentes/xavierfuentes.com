# LinkedIn Post Template

Copy to `content/linkedin/YYYY-MM-slug-linkedin.md`

## Frontmatter

```yaml
---
idea_id: "YYYY-MM-your-idea-slug"
pillar: technology-strategy
status: draft
sequence: 1  # Optional: if part of series
target_audience: cto_startup_scaleup
target_outcome: authority
---
```

## File Structure

Each file contains 2-3 posts separated by `---`:

```markdown
# Post 1: Framework (Tuesday)

[Hook under 140 chars]

[Body - 2-3 short paragraphs]

[Question or soft CTA]

---

# Post 2: Industry Take (Wednesday)

[Hook under 140 chars]

[Body - contrarian view with substance]

[Question to spark discussion]

---

# Post 3: Personal Lesson (Thursday)

[Hook - vulnerable opening]

[Story - what happened, what you learned]

[Takeaway + question]
```

## Post Templates

### Tuesday: Framework

```
[Contrarian hook about common practice]

Here's the framework I use instead:

1. [Step one with why]
2. [Step two with why]
3. [Step three with why]

The key insight: [one-line summary]

What framework do you use for [topic]?
```

### Wednesday: Industry Take

```
[Bold statement that challenges consensus]

Most people think [common belief].

But after [experience/data], I've learned:

[Your position with evidence]

The real issue is [deeper insight].

Agree or disagree?
```

### Thursday: Personal Lesson

```
[Vulnerable hook about a mistake/lesson]

[X] years ago, I [what happened].

What I learned:
- [Lesson 1]
- [Lesson 2]
- [Lesson 3]

If I could go back, I'd [key advice].

What's a lesson you wish you'd learned earlier?
```

## Frontmatter Reference

| Field | Required | Notes |
|-------|----------|-------|
| idea_id | Yes | Link to source idea |
| pillar | Yes | From idea |
| status | Yes | draft / ready_for_review / ready_for_posting / posted |
| sequence | No | For multi-part series |
| target_audience | No | From idea |
| target_outcome | No | From idea |
