# Blog SEO Guidelines

## Primary Keywords (High Intent)

- "fractional CTO services"
- "startup technology strategy"
- "engineering team scaling"
- "technical due diligence"

## Secondary Keywords (Thought Leadership)

- "technology leadership frameworks"
- "startup CTO responsibilities"
- "engineering management best practices"
- "technical debt strategy"

## On-Page SEO

### Title
- Include primary keyword naturally
- Keep under 60 characters
- Make it compelling, not just keyword-stuffed

### Meta Description
- 150-160 characters
- Include primary keyword
- Compelling hook that encourages click
- End with implicit or explicit CTA

**Example:**
```
Most CTOs treat technical debt like it's optional. Here's the framework I use to make it a strategic assetâ€”and get buy-in from the board.
```

### First Paragraph
- Include primary keyword in first 100 words
- Hook the reader immediately
- Set up the problem or contrarian take

### Headings
- Use H2 for main sections
- Use H3 for subsections
- Include secondary keywords naturally
- Make headings scannable and informative

### Internal Links
- Link to 2-3 related posts where relevant
- Use descriptive anchor text (not "click here")
- Link to pillar content where appropriate

## Frontmatter SEO Fields

```yaml
meta_title: "SEO-optimised title (if different from title)"
meta_description: "150-160 char compelling description"
seo_keyword: "primary target keyword"
```

## Content Length for SEO

| Type | Length | SEO Value |
|------|--------|-----------|
| Framework/how-to | 1,500-2,000 | High (evergreen) |
| Thought leadership | 1,800-2,200 | High (authority) |
| Case study | 2,000-2,500 | Medium (specific) |
| Opinion/hot take | 800-1,200 | Lower (engagement) |
