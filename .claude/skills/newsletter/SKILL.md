# Newsletter Skill (The Jungle Brief)

Create curated, high-signal newsletter issues that build subscriber loyalty.

## When to Use

- Assembling The Jungle Brief issues from canonical ideas
- Creating newsletter sections (deep insight, template/tool, links)
- Reviewing or editing newsletter drafts
- Planning newsletter content calendar

## Quick Reference

**Frequency:** Bi-weekly (moving to weekly)

**Structure per issue:**
1. One deep insight (from a canonical idea)
2. One template or tool
3. 3-5 curated links with commentary

**Target length:** 5-7 minute read (~1,200-1,500 words)

- `context/voice.md` - Newsletter-specific voice (curated, direct)
