# Newsletter Issue Template

Copy to `content/newsletter/YYYY-MM-DD-issue-NN.md`

## Frontmatter

```yaml
---
idea_id: "YYYY-MM-your-idea-slug"  # Primary idea this issue explores
issue_number: NN
status: draft
pillar: technology-strategy
target_audience: cto_startup_scaleup
target_outcome: newsletter_engagement
subject_line: "<50 chars, hooks the insight>"
---
```

## Issue Structure

```markdown
[Opening hook - one bold statement, no greeting]

---

## [Deep Insight Title]

[Context - why this matters now, 2-3 sentences]

[Core insight - the thing you're teaching, 3-4 paragraphs]

[Example or story - make it concrete, 2-3 paragraphs]

**The takeaway:** [One actionable sentence]

---

## Quick Tool: [Name]

[Brief context - 2-3 sentences max]

[The actual thing - table, checklist, framework]

[How to use it - 1-2 sentences]

---

## What I'm Reading

**[Article Title 1]** by [Author]
[Your 1-2 sentence take on why it matters]

**[Article Title 2]** by [Author]
[Your 1-2 sentence take on why it matters]

**[Article Title 3]** by [Author]
[Your 1-2 sentence take on why it matters]

---

[Sign-off sentence]

Xavi
```

## Frontmatter Reference

| Field | Required | Notes |
|-------|----------|-------|
| idea_id | Yes | Link to primary source idea |
| issue_number | Yes | Sequential issue number |
| status | Yes | draft / ready_for_review / scheduled / sent |
| pillar | No | From primary idea |
| target_audience | No | From primary idea |
| target_outcome | No | Usually newsletter_engagement |
| subject_line | Yes | Email subject, <50 chars |

## Subject Line Patterns

**Good:**
- "The technical debt denial problem"
- "A framework for saying no"
- "What I learned from 50 tech stack reviews"

**Avoid:**
- "The Jungle Brief #14"
- "This week's newsletter"
- "5 tips for..."

## Quality Checklist

Before sending:

- [ ] Opening hook is under 140 characters
- [ ] Deep insight has clear takeaway
- [ ] Template/tool is immediately usable
- [ ] All links have commentary (not just listed)
- [ ] Total read time is 5-7 minutes
- [ ] Subject line hooks without clickbait
- [ ] No greeting, just value from line one
